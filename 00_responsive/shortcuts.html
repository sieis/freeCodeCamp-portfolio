Prerequisites

This will help you the most if you already know your way around some or all of the programs I am highlighting.

This documentation is for those comfortable using these pieces of software and interested in becoming more proficient and efficient.

Gmail Shortcuts

Quick! Go turn on keyboard shortcuts in your Gmail. I promise you will not regret it.

Here is a link to the directions, and the full list of shortcuts. I use these all day every day...well, at least while I'm using Gmail. Navigation just got a lot easier, and the act of composing and replying to messages is a keystroke away now:

  • Go to next page: g + n
  • Go to next Inbox section: ` (at the top of the keyboard under esc
  • Go to previous Inbox section: ~ (same key at the top under esc, just hold shift
  • Compose new message: c
  • Reply to a message: r

Google Sheets Shortcuts

Hiding rows and columns is simple, but time-saving. Here is a link to all of the shortcuts available.

  • To hide a row: Ctrl + Alt + 9
  • To hide a column: Ctrl + Alt + 0
  • To unhide a row: Ctrl + Shift + 9
  • To unhide a column: Ctrl + Shift + 0

VS Code Shortcuts

Here is a list to a ton of VS Code shortcuts. A couple examples I use all the time:

  • To delete a line: Ctrl + Shift + K
  • To move a line (or multiple lines selected) up or down: alt + ↑ or alt + ↓
  • To select all occurrences of current selection: Ctrl + Shift + L
Notion Shortcuts

Here's Notion's shortcut page with a ton of shortcuts.

Their slash section is particularly useful in quickly creating content with different types of blocks:

  • new text block: /text
  • new bulleted list: /bullet
  • create different sized headings: /h1, h2, or /h3
